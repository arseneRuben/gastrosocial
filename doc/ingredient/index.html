GET/ingredients?page=idPage&orderBy=categories,createdAt,updatedAt,title&favorite=true

Liste paginee des ingredients ordonnees par attributs

Paramettres de la requette

parametres Types Requis Description
idPage Entier OUI Numero de page du resultat de la requete
createdAt DateTime NON Date de creation des ingredients
updatedAt DateTime NON Date de derniere modification des ingredients
title String NON Titre validee des ingredients
createdAt DateTime NON Date de creation des ingredients

Exemple de reponse

         
             $ {
                   
                    "adoptedTitle": "Tomate",
                    "createdAt":"2021-08-06T17:04:26.000Z"
                    "fullName": 4,
                    "updatedAt":"2021-08-06T17:04:26.000Z"

                }

Details de reponse

attribut Description Requis
fullName Nom du createur de l'ingredient OUI
createdAt Date de creation de l'ingredient NON
updatedAt Date de mise a jour de l'ingredient NON
adoptedTitle Titre valide de l'ingredient OUI
